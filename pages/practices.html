Учебные и научные практики

Научно-исследовательская практика

Проходила на кафедре экономической теории под руководством профессора Иванова А.И. Тема исследования: "Анализ эффективности цифровых платформ для малого бизнеса в период экономических санкций". В рамках практики проводился сбор и анализ данных по 150 компаниям малого бизнеса, использовались методы регрессионного анализа и машинного обучения для выявления ключевых факторов успеха.

Результат: Подготовлена научная статья для журнала "Вопросы экономики", разработана модель прогнозирования адаптационного потенциала предприятий.

Производственная практика

Практика в Департаменте стратегического анализа Центрального банка РФ. Участие в проекте по анализу устойчивости банковской системы к киберрискам. Основные задачи: анализ международного опыта регулирования, моделирование сценариев кибератак на финансовые институты, подготовка аналитических записок для руководства.

Результат: Разработана методика оценки киберрисков для банков разного масштаба, рекомендации вошли в проект новых нормативных документов ЦБ.

Педагогическая практика

Проведение семинаров по курсу "Эконометрика" для студентов 2 курса бакалавриата. Разработка учебных материалов, проведение консультаций, проверка домашних заданий. Подготовил и провел серию мастер-классов по работе с программой R для экономического анализа.

Результат: Разработаны интерактивные учебные материалы, которые были рекомендованы кафедрой для использования в следующем учебном году.
